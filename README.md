## Project Name
GSM Based Fire Alert System

## Project Scope
The scope of this project is to develop an automatic fire detection and alert system using GSM technology. It continuously monitors environmental conditions such as smoke and temperature and sends instant alert messages to users during fire emergencies. The system enhances safety in homes, offices, industries, and public places.

## Project Description
The GSM Based Fire Alert System uses a microcontroller along with smoke and temperature sensors to detect fire hazards. When the detected smoke level or temperature exceeds a predefined threshold, the system activates a buzzer and sends an SMS alert through the GSM module to the concerned person. This allows quick action to be taken, reducing the risk of property damage and loss of life.
